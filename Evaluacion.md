# 🧪 Evaluación Examen Backend

📅 **Fecha:** 2026-03-21
⏱️ **Duración:** 8:40 - 10:20
🎯 **Valor total:** 3 puntos

---

## 📊 Criterios de Evaluación (5 puntos base)

Cada criterio vale **1 punto**.
La calificación final se convierte a una escala de **3 puntos**.

---

### ✅ 1. CRUD funcionando (1 pt)

* ✔️ Puede crear, listar, actualizar y eliminar alumnos
* ❌ Si falta alguna operación → 0.5 o 0

---

### ✅ 2. Filtro por fecha (1 pt)

* ✔️ Funciona correctamente con formato `YYYY-MM-DD`
* ❌ No funciona o formato incorrecto → 0

---

### ✅ 3. Login (1 pt)

* ✔️ Endpoint funcional
* ❌ Implementación incorrecta o hardcode deficiente → 0.5 o 0

---

### ✅ 4. Estructura y orden (1 pt)

* ✔️ Separación básica (rutas, controladores, modelos)
* ❌ Todo en un solo archivo o mala organización → 0

---

### ✅ 5. Swagger + JSON + errores (1 pt)

* ✔️ Documentación en Swagger y respuestas en JSON correctas
* ❌ Falta Swagger → 0 directo en este criterio

---

## 🧮 Cálculo de Calificación Final

Se utiliza la siguiente fórmula:

```
(puntos_obtenidos / 5) * 3
```

### 📌 Ejemplos:

* 5/5 → 3.0
* 4/5 → 2.4
* 3/5 → 1.8

---

## ⚡ Forma de Evaluación

* Se probará la API (Postman o navegador)
* Se validará funcionalidad principal
* Se revisará estructura básica del código
* Se verificará documentación (Swagger)

⏱️ Tiempo estimado por alumno: **3 a 5 minutos**

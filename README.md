# 🧪 Examen Técnico Backend

📅 **Fecha:** 2026-03-21 realizacion de 8:40 a 10:20

## 🎯 Objetivo
Desarrollar una API REST que permita la gestión de alumnos.

---

## 📌 Requerimientos

- Implementar un CRUD (Create, Read, Update, Delete) de alumnos.

- Alta de alumnos con los siguientes campos mínimos:
  - nombre  
  - apellido paterno  
  - apellido materno  
  - matrícula  
  - correo  

- Crear un endpoint para consultar alumnos por rango de fecha de alta:
  - Formato de fecha: `YYYY-MM-DD`
  - Ejemplo:
    ```
    2026-03-25 - 2026-03-26
    ```

- Implementar un endpoint de autenticación:
  - Login con usuario y contraseña

- Todo con documentacion en swagger

---

## ⚙️ Consideraciones

- La API debe manejar respuestas en formato JSON  
- Manejo básico de errores  
- Estructura organizada del proyecto (rutas, controladores, modelos)
- Montar en en repositorio de github
